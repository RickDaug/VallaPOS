# VallaPOS Starter

A simple browser-based POS starter for food trucks, mobile vendors, barbers, lawn care, and small service businesses.

## Tagline

**No hardware contract. No complicated setup. Just log in and sell.**

## MVP Included

- Browser checkout screen
- Product/menu grid
- Cart with tax, tip, discounts, and total
- Mark-as-paid workflow
- Dashboard cards
- Orders list
- Inventory list
- Customer list
- Employee role preview
- Starter database schema

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Prisma
- PostgreSQL-ready schema

## Quick Start

```bash
npm install
npm run dev
```

Open:

```bash
http://localhost:3000
```

## Product Positioning

VallaPOS is designed for operators who do not want a complicated enterprise POS system. The first niche should be mobile and local businesses that need a fast checkout tool on any phone, tablet, laptop, or desktop browser.
